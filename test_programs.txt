Example program (Add 3 and 2, output):
ld r1, 3 ; set r1 = 3
add r1, 2 ; add r1 by 3 (equals 5)
out 0x1 ; output to screen

in hex:
0x3 0x1 0x3 (load)
0x1 0x1 0x2 (add)
0xC 0x1 (output)
0x3 0x2 0x8 (load)
0x5 0x1 0x2 (compare r1 and r2)
0x4 0x0 (jmp)
